URL : https://expressjs.com/en/starter/generator.html

This tool is used to create express applications

steps to configure express generator tool and create
new express app

->install express generator

npm install -g express-generator

->create express app for specified view engine

express --view=ejs myapp

->move to current app

->install dependencies

npm install

->run app

set DEBUG=myapp:* & npm start
